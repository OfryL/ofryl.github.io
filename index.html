About

It started with a soldering iron and a pile of components I probably shouldn’t have plugged in. RF circuits, microcontrollers, code that talks to hardware — I just wanted to know how everything worked, and then how to make it do something new.

That itch never went away. I still build little things for the fun of it — Telegram bots, browser plugins, weekend experiments — chasing whatever technology I haven’t taken apart yet. Lately that’s AI: wiring up LLMs, building agents and MCP servers, and poking at what these models can actually do. Same curiosity, new pile of parts.

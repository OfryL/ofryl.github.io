It started with a soldering iron and a pile of components I probably shouldn't have plugged in — RF circuits, microcontrollers, code that talks to hardware. That itch never went away. These days I'm **Platform Engineering Lead at [Tastewise](https://tastewise.io)**, and previously a **founding engineer at Autofleet** (seed → acquisition, 2024).

- 🔭 Building AI platform infrastructure — MCP servers, LLM integration, and enterprise SSO
- 🌱 Wiring up LLMs and agents, and poking at what these models can actually do
- ⚡ Still build little things for fun: Telegram bots, browser plugins, weekend hardware experiments
- 📫 Reach me at **ofryli@gmail.com** or on [LinkedIn](https://www.linkedin.com/in/ofry-linkovsky/)

Ofry Linkovsky — Portfolio

Personal portfolio site, served via GitHub Pages at https://ofryl.github.io

Static site — no build step, no JavaScript. index.html is plain HTML with
inline CSS. Edit index.html and push to master to deploy.

Files:
  index.html        The whole site — hero, about, experience, projects, contact.
  404.html          Custom not-found page.
  images/           Site images (avatar.png and social icons).
  llms.txt          AI-readable site map (see https://llmstxt.org).

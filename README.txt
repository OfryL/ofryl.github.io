Ofry Linkovsky — Portfolio

Personal portfolio site, served via GitHub Pages at https://ofryl.github.io

Static site — no build step. index.html is self-contained (HTML, CSS, and JS
inline, images embedded as base64). Edit index.html and push to master to deploy.

Files:
  index.html   The whole site — JS-rendered hero, experience, projects, contact.
  404.html     Custom not-found page.
  llms.txt     AI-readable site map (see https://llmstxt.org).
